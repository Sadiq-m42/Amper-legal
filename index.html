سياسة الخصوصية لتطبيق Spark

تاريخ التحديث: يونيو 2026

يلتزم تطبيق Spark بحماية خصوصية بياناتكم الإدارية والمالية. توضح هذه السياسة كيفية التعامل مع البيانات الصادرة والواردة داخل النظام.

1. البيانات التي نجمعها

  • بيانات الحساب: الاسم، البريد الإلكتروني، ورقم الهاتف عند إعداد حسابات الموظفين والمشتركين.
  • البيانات المالية والتشغيلية: سجلات الجباية، النثرية، المبالغ المستلمة، وصرفيات الكاز لإدارة المولدات الكهربائية.

2. صلاحيات الأجهزة (Permissions)

  • صلاحية البلوتوث والموقع (Android): تُستخدم حصراً للاقتران بالطابعات الحرارية المحمولة وطباعة وصولات القبض للمشتركين ميدانياً.
  • المصادقة الحيوية (Face ID / Touch ID): تُستخدم محلياً على جهازك لتأمين شاشات الأرباح والصيرفة ولا يتم رفعها أو تخزينها على سيرفراتنا.

3. مشاركة وحماية البيانات

يتم تخزين البيانات بشكل آمن عبر سحابة Firebase المشفرة. لا نقوم ببيع، مشاركة، أو تسريب أي بيانات مالية أو شخصية لأي جهات خارجية أو إعلانية.

4. حذف الحساب والبيانات

يمكن للموظفين والمسؤولين طلب حذف حساباتهم وبياناتهم بالكامل فوراً من خلال الانتقال إلى شاشة "الملف الشخصي -> إدارة الحساب" داخل التطبيق، أو عبر مراسلتنا مباشرة على البريد الإلكتروني المعتمد.

5. التواصل والدعم

لأي استفسار قانوني أو تقني، يرجى مراسلتنا عبر: support@spark.app
